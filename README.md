### Requirements
* gems:
    * **rails** >= 6
    * **devise** >= 4
    * **simple_form**
* npm packages:
    * **bootstrap** >= 5

### Installation
* clone repo
* `cd` into directory
* run `bundle`
* migrate database with run `rails db:migrate`

### Running
* `rails s` to start server
